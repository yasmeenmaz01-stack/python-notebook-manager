# Personal Notebook Manager

A Python console application for managing personal notes.  
The program allows users to add, view, search, edit, delete, and save notes using a JSON file.

## Features
- Add new notes
- List all notes
- Search notes by keyword (title or content)
- Filter notes by tag
- Edit existing notes
- Delete notes
- Save notes to a JSON file
- Load notes automatically on startup


## How to Run:
1.Open PyCharm on your computer.
2.Click File → Open and select the project folder that contains the file notebook.py.
3.Inside PyCharm, locate the file notebook.py in the left project panel.
4.Open the file by clicking on it.
5.Run the program by clicking the green Run ▶ button at the top-right corner of PyCharm (or right-click the file and choose Run notebook_manager.py).
6.The program will now start in the PyCharm console, where you can use the menu to add, list, search, edit, and delete notes.
##Note:
The file notes.json will be created automatically the first time you save a note, so you don’t need to create it manually.
